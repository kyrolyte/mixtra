# Mixtra - A Minimal Hugo Theme

**Mixtra** is a lightweight, clean, and opinionated Hugo theme designed for those who value speed and simplicity.

By default, it renders standard Markdown with a minimalist aesthetic, stripping away unnecessary overhead to ensure your content takes center stage. While it excels as a "vanilla" blogging theme, it is built with a modular structure, making it the perfect springboard for developers who want a solid, lean foundation to extend and customize as their site grows.
